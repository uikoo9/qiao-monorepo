# offline-to-online
offline to online on nodejs and browser

## install
```
npm install offline-to-online
```

## api
### offline to online on browser
```javascript
'use strict';

var q = require('qiao-is-online-browser');

q.offlineToOnline(function(){
    console.log('offline-to-online');
});
```
