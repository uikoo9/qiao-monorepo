## cli

### dc

```shell
# 列出文件夹下所有npm包上一个月的下载量
qlerna dc ./packages
```

### ncu

使用[ncu](https://www.npmjs.com/package/npm-check-updates)检查文件夹下所有 npm 包是否有更新

```shell
qlerna ncu ./packages
```

### pkg

```shell
# 列出文件夹下所有npm包的dependencies
qlerna pkg ./packages

# 列出文件夹下所有npm包的devDependencies
qlerna pkg ./packages dev
```
