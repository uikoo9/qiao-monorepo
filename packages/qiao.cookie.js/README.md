浏览器 Cookie 相关操作封装，详见：[cookie 文档](https://developer.mozilla.org/zh-CN/docs/Web/API/Document/cookie)

## api

### cjs

```javascript
// q
const q = require('qiao.cookie.js');

// get
q.get('key');

// set
q.set('key', 'value');

// del
q.del('key');

// has
q.has('key');

// keys
q.keys();
```

### mjs

```javascript
// q
import { get, set, del, has, keys } from 'qiao.cookie.js';

// get
get('key');

// set
set('key', value);

// del
del('key');

// has
has('key');

// keys
keys();
```
